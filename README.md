# 🚀 Project Management App

A full-stack Project Management application to manage tasks and projects efficiently.

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📁 Create Projects
* 📋 View Project List
* ✅ Update Project Status (Pending / Completed)

---

## 🛠 Tech Stack

### Frontend:

* React (Vite)

### Backend:

* Node.js
* Express

### Database:

* MongoDB (Coming Soon 🚧)

---

## 📂 Project Structure

```
Project_Management/
 ├── backend    # Node.js + Express API
 ├── src        # React frontend
 ├── package.json
```

---

## 📦 Installation & Setup

### 1️⃣ Clone repository

```bash
git clone https://github.com/amitvishwakarma8881/Project_Management.git
cd Project_Management
```

### 2️⃣ Install frontend dependencies

```bash
npm install
npm run dev
```

### 3️⃣ Run backend

```bash
cd backend
npm install
node server.js
```
